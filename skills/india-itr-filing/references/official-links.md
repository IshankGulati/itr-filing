  - current common path: `Login > AIS > TIS`

## Official download assets for JSON building

Use the Downloads page for the current `AY` to access:

- common offline utility for `ITR-1`, `ITR-2`, `ITR-3`, `ITR-4`
- separate utility stream for `ITR-5` to `ITR-7`
- form-specific utility
- JSON schema
- schema change document
- validation document

The official offline-utility manual also explains how to:

- download the utility
- download pre-filled data JSON
- import pre-filled JSON into the utility
- generate or submit the prepared return through the utility workflow

Do not validate a JSON payload against an old schema when the current AY page shows a newer release date.
